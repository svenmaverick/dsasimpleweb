Pseudocode List

1. Find Maximum

max = A[0]
for i = 1 to n-1
    if A[i] > max
        max = A[i]
return max

2. Sum of Array

sum = 0
for each x in A
    sum = sum + x
return sum

3. Reverse Array

left = 0, right = n-1
while left < right
    swap A[left], A[right]
    left++, right--

4. Count Evens

count = 0
for x in A
    if x mod 2 == 0
        count++
return count

5. Palindrome

left = 0, right = n-1
while left < right
    if A[left] != A[right]
        return false
    left++, right--
return true

6. Swap Values

temp = a
a = b
b = temp

7. Power

result = 1
for i = 1 to n
    result = result * x
return result

8. Minimum

min = A[0]
for i = 1 to n-1
    if A[i] < min
        min = A[i]
return min

9. Print 1 to N

for i = 1 to n
    print i

10. Average

sum = 0
for x in A
    sum = sum + x
return sum / n
